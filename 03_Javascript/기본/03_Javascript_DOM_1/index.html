직접선택자(원거리선택자)

#1. id 선택자 : 태그 내부에 존재하는 id값을 기준으로 선택

#2. 클래스 선택자 : 여러가지 클래스를 보유하고 있는 모든 태그를 선택. 배열리스트로 확보 => 인덱스 번호가 존재

#3. 태그 선택자 : 여러가지 태그를 모두 선택. 배열리스트로 확보 => 인덱스 번호가 존재 (문서상에 존재하는 body 라는 태그도 동일)

#4. 특정명칭을 기반으로 선택(querySeletor) : CSS상에서 선택하였던 모든 방식으로 지정 가능. 단일 선택자

#5. 특정명칭을 기반으로 다수를 선택(querySeletorAll) : CSS상에서 선택하였던 모든 방식으로 지정 가능. 다수를 선택하기에 배열리스트로 확보 => 인덱스 번호가 존재
