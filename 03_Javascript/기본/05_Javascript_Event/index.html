이벤트

이벤트란, 사용자가 일련의 동작을 수행하는 과정에서 함수문을 호출하여 실행한 후 결과로써 피드백을 받는 과정

- 요소 대상 이벤트 : 클릭(click), 마우스(mouse), 키보드(keyboard)

- 전체 대상 이벤트 : load, resize, scroll

- form 대상 이벤트 : submit, change

이벤트를 진행할 수 있는 함수호출 방법 - 외부스크립트 연동시

1. 태그 내부에 on이벤트명="함수호출문();"

2. (in Javascript) 선택자.on이벤트명 = 함수호출문

3. (in Javascript) 선택자.addEventListener("이벤트명", function(){ 함수 실행문 })

요소 대상 이벤트 : mouseover & mouseout

#1. 직접 함수 호출

#2. 외부 자바스크립트에서 이벤트 실행 (on이벤트명 적용)

#3. 외부 자바스크립트에서 이벤트 실행 (addEventListener 적용)

요소대상이벤트 : click

요소 대상 이벤트: keyboard 이벤트

keydown: 키보드가 눌렸을때 이벤트가 발생한다.

keyup: 키보드가 눌렸다가 떼었을 떄, 이벤트가 발생한다.

keypress: 키보드가 누른 상태가 되었을 때 이벤트가 발생한다.
