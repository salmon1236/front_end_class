Padding 작성시, padding: 20px; 상우하좌(시계방향)의 padding 값이 동일한 경우 스타일 적용시 사용

padding: 20px

Padding작성 시, Padding-top: 30px; Padding-right: 20px; Padding-bottom: 40px; Padding-left: 10px; 으로 각각 다른 값으로 스타일을 부여하고 싶다면, 상단부터 시계방향으로 값을 넣음.
padding: 30px 20px 40px 10px; padding: 상 우 하 좌;

padding: 30px 20px 40px 10px

Padding작성 시, padding-top: 20px; padding-right: 40px; Padding-bottom: 20px; Padding-left: 40px; 으로 스타일을 부여하고 싶다면 통합시 상단으로부터 시계방향으로 각각 방위에 대한 값을 부여 가능하되 통합도 가능하다.
padding: 20px 40px 20px 40px; padding: 상 우 하 좌;
padding: 20px 40px; padding: (상하) (우좌);

padding: 20px 40px

Padding작성 시, padding-top: 60px; padding-right: 40px; Padding-bottom: 20px; Padding-left: 40px; 으로 스타일을 부여하고 싶다면 통합시 상단으로부터 시계방향으로 각각 방위에 대한 값을 부여 가능하되 통합도 가능하다.
padding: 60px 40px 20px 40px; padding: 상 우 하 좌;
padding: 60px 40px 20px; padding: 상 (우좌) 하;

padding: 60px 40px 20px
